# Animações Lottie do Pódio (aba Métricas)

Coloque aqui os arquivos `.json` de animação (Lottie). Cada métrica do Pódio usa um
arquivo com nome fixo. Enquanto o arquivo não existir, o app mostra automaticamente
uma ilustração SVG animada de fallback (nenhum erro).

## Onde conseguir

- https://lottiefiles.com — use animações com **licença livre (Lottie Simple License / free)**.
- Baixe em **formato JSON** (não `.lottie`), botão "Download" → "Lottie JSON".
- Renomeie exatamente para o nome abaixo e salve nesta pasta.

## Arquivos esperados (nome exato)

| Arquivo | Métrica | O que procurar na LottieFiles |
|---|---|---|
| `leads.json` | Leads Captados | "magnet attract", "magnet leads", "lead generation" |
| `patients.json` | Pacientes Captados | "calendar appointment", "booking", "schedule check" |
| `sales.json` | Novas Consultas Realizadas | "trophy", "success", "goal achieved" |
| `scheduling.json` | Taxa de Agendamento | "funnel", "calendar", "conversion funnel" |
| `conversion.json` | Taxa de Conversão | "funnel conversion", "percentage", "growth" |
| `revenue.json` | Faturamento | "money", "coins", "revenue growth" |
| `investment.json` | Investimento (Google/Meta) | "megaphone", "advertising", "marketing" |
| `roas.json` | ROAS | "growth chart", "profit", "arrow up" |
| `ticket.json` | Ticket Médio | "price tag", "pricing", "discount tag" |
| `cost.json` | Custos (por Lead / Agend. / Conversão) | "target", "cost", "budget" |
| `lost.json` | Perdidos (opcional) | "lost", "declined", "sad" |

## Dicas

- Prefira animações com **fundo transparente** e proporção mais ou menos quadrada.
- Cores fortes de destaque funcionam melhor no card branco.
- Tamanho ideal por arquivo: até ~150 KB (animações vetoriais leves).
- Depois de adicionar, é só rodar o app: a animação aparece no lugar do fallback.
